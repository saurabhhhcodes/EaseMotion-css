# 3D Card Tilt

CSS-only 3D card tilt effect with perspective, rotation, and glare on hover.

## Features
- 3D perspective transform on hover
- Gradient border with glassmorphism background
- Glare/reflection overlay effect
- Icon lift with deeper Z translation on hover
- Smooth spring-like transitions
- 4 cards with varied tilt angles
- Dark theme optimized

## Usage
Include `style.css` and use the BEM-structured HTML from `demo.html`.

## Browser Compatibility
Modern browsers supporting CSS 3D Transforms, preserve-3d, and Custom Properties.
