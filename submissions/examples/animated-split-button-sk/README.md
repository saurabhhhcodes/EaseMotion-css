# Animated Split Button

A CSS-only split button component with primary action and dropdown arrow parts. Hover and active states on both segments. Multiple examples with different labels.

Created by @saurabhhhcodes

## Features
- Primary action + dropdown toggle separated by a divider
- Independent hover/active states on both halves
- Multiple label variants (Save, Delete, Edit, Share)
- Pure CSS using adjacent sibling selectors for hover chaining
- BEM naming convention
- Inter font

## States
- Default — flat appearance
- Hover — subtle lift and background shift
- Active (mousedown) — inset shadow press effect
