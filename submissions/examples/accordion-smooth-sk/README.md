# Smooth Accordion

CSS-only accordion with animated expand.

## Usage

```html
<details class="acc" open><summary>What is EaseMotion?</summary><p>Human-readable CSS animation utilities.</p></details><details class="acc"><summary>How do I use it?</summary><p>Copy classes into your markup.</p></details>
```

## Why EaseMotion CSS

Pure CSS animation utility — no JavaScript required.
