# Animated Tag Cloud

CSS-only tag cloud with skill badges, hover glow, and size variations.

## Features

- Pill-shaped tags with gradient backgrounds
- Five size variants (sm, md, lg, xl)
- Ten color themes (indigo, blue, pink, green, purple, orange, teal, red, cyan, amber)
- Hover scale and glow effect on each tag
- Dark theme optimized background
- Gradient title with text clipping
- Custom properties for easy theming

## Usage

Include the `style.css` and use the BEM classes. Tags wrap naturally with `flex-wrap`. Add `tagcloud__tag--{size}` and `tagcloud__tag--{color}` modifiers to each tag.

## Browser Support

All modern browsers (Chrome, Firefox, Safari, Edge). Uses CSS gradients, custom properties, transforms, and box-shadow.
