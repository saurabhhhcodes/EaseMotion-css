# Animated Color Picker

CSS-only color swatch picker with selection animation, checkmark indicator, and live preview.

## Features
- Grid of color swatches with smooth selection
- Scale + ring animation on selected swatch
- White checkmark indicator on selected color
- Live preview bar showing selected color and hex value
- Focus-visible accessibility
- Hover scale effect
- Responsive grid layout

## Usage
Include `style.css` and use the BEM-structured HTML from `demo.html`.

## Browser Compatibility
Modern browsers supporting CSS Grid, Transforms, and :checked/:focus-visible pseudo-classes.
